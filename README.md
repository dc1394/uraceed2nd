★これは何？
　裏CEED第2回オンラインセミナーの「有限要素法の紹介」で使用した、FEMでd^2f(x)/dx^2 = 1という2階線形常微分方程式を解くコードです。有限要素離散化の際の計算のPDFも同梱しています。
